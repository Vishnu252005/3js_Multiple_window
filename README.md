# AI & Data Science Projects Portfolio

This repository contains a collection of applied machine learning and data science projects focused on sustainability, smart systems, and social impact. Each notebook demonstrates a complete workflow, from data preprocessing to model evaluation and ethical reflection.

---

## 📦 Projects Overview

### 1. AI-Driven Smart Waste Segregation
- **Theme:** Environmental Intelligence & IoT
- **SDG Goals:** SDG 12 (Responsible Consumption), SDG 15 (Life on Land)
- **Summary:**  
	Implements Decision Tree, Naive Bayes, Logistic Regression, and ANN to classify waste types (Organic, Plastic, Metal, E-waste) using a Kaggle dataset. Compares models on accuracy, F1 score, and inference time. Includes visualizations and a discussion on the ethical impact of automation on low-income waste workers.
- **Colab:** [Open in Colab](https://colab.research.google.com/github/Vishnu252005/3js_Multiple_window/blob/main/AI_Driven_Smart_Waste_Segregation.ipynb)

---

### 2. Crop Pattern Clustering for Sustainable Farming
- **SDG Goals:** SDG 2 (Zero Hunger), SDG 12 (Responsible Consumption)
- **Summary:**  
	Uses K-Means and Expectation-Maximization (Gaussian Mixture Models) to cluster crop patterns from the FAO Crop Production Dataset. Evaluates clusters using Silhouette Score, BIC, and AIC. Discusses the importance of equitable access to agri-analytics tools for small farmers.
- **Colab:** [Open in Colab](https://colab.research.google.com/github/Vishnu252005/3js_Multiple_window/blob/main/crop_clustering.ipynb)

---

### 3. Smart Building Energy Efficiency Classification
- **Summary:**  
	Demonstrates a machine learning workflow for classifying building energy efficiency using the UCI Energy Efficiency Dataset. Compares SVM, Decision Tree, and ANN models with cross-validation, confusion matrices, ROC curves, and feature importance analysis.
- **Colab:** [Open in Colab](https://colab.research.google.com/github/Vishnu252005/3js_Multiple_window/blob/main/smart_build_energy_efficiency.ipynb)
